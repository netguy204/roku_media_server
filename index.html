roku_media_server by netguy204

A client and server combo for serving mp3s and other local media over your local network to your Roku set-top box.

Dependencies

Python

Install

# Download and install python for your platform # Download MyMedia # Put your Roku into developer mode # Run springboard.vbs to install the channel on your Roku and start the server
